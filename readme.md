Examlpe:

n                                              = 11 in base-10 number system
    8     +     0     +     2     +     1      = 11
     2^3              +      2^1  +      2^0   = 11
(1 * 2^3) + (0 * 2^2) + (1 * 2^1) + (1 * 2^0)  = 11
1         0         1         1
binary(n) = 1011 in base-2 numeral system      = 11 in base-10 number system
bin(n)    = 0b1011
